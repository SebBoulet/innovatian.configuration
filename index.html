Innovatian.configuration

Configuration library supporting INI, XML, Isolated Storage, Registry, command line, .NET config files as storage mechanisms.

Innovatian.Configuration.

Innovatian.Configuration was inspired by the Nini project: http://nini.sourceforge.net/ This project tries to accomplish what Nini does but implemented using newer technology. Most features are supported in about 25% the lines of code and half the library size.
